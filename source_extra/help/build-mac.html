Building for OS X

This information pertains to building Diamond for OS X.

Xcode

Xcode is an IDE that contains software development tools for OS X. The current version is 4.5 and is available for free at the App Store. There are several GNU tools in Xcode that are used for building Diamond. Xcode should be installed on your Mac OS X development computer. To download Xcode click on the following link Xcode

Starting around version 4.3, Xcode no longer automatically installs some required components. Once Xcode has been installed, you will need run Xcode and install the command line tools by hand.

  • Run Xcode
  • Select Xcode (from the menu) and then click on Preferences
  • Select Downloads, Components tab
  • Click the Install button to the right of the 'Command Line Tools'

Compiler & Autotools

Xcode does not install Autotools, which is required to build the Diamond Editor. Use MacPorts or the equivalent to install Autotools.

The Diamond Editor requires Apple Clang version 4.2 or newer.

MacPorts can be downloaded and installed from MacPorts

The following is a straight forward guide for installing MacPorts MacPorts Guide

Using MacPorts, install the ports listed below. MacPorts will automatically adjust your PATH for Automake and Autoconf.

  • sudo port install automake
  • sudo port install autoconf

Compiling with Clang will require the following environment variables.

  • CC=clang
  • CXX=clang++
  • OBJCXX=clang++

Alternatively, you can create a file like 'config-opt' and pass this file name to configure when you build Diamond. Here is a sample config-opt file.

CC=clang CXX=clang++ OBJCXX=clang++ --prefix $HOME/cs_lib
