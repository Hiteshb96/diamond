Building for Windows

This information pertains to building Diamond to run on Windows.

Diamond was built using Autotools. In order to use Autotools several programs and utilities must be installed.

Note: Windows does not provide a Bourne shell as a standard component. In order to build Diamond this shell program is required to run the configure scripts. It will be installed as part of Msys.

Compiler

Diamond 32-bit was built using version 4.7.2 of MinGW. Diamond 64-bit was built using version 4.8.1 of MinGW.

Version 4.7.2 released on February 11, 2013

MinGW 32-bit Version x32-4.7.2-release-posix-sjlj-rev9.7z   MinGW 32

MinGW 64-bit Version x64-4.7.2-release-posix-sjlj-rev9.7z   MinGW 64


Version 4.8.1 released on August 30, 2013

MinGW 32-bit Version x32-4.8.1-release-posix-sjlj-rev5.7z   MinGW 32

MinGW 64-bit Version x64-4.8.1-release-posix-sjlj-rev5.7z   MinGW 64

Utilities

We recommend installing this version of MSYS, which provides GNU Make and other tools designed to work with the MinGW version of GCC. There is only one version of MSYS for 32-bit and 64-bit systems.

MSYS Version msys+7za+wget+svn+git+mercurial+cvs-rev9.7z MSYS
