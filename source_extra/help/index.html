Introduction

Diamond is a full editor capable of running on Windows, Linux, and OS X. The Diamond project began as a replacement for the Crimson and Emerald Editors which were available only for Windows.

For OS X, Diamond is an alternative to TextWrangler.

Diamond is a new project designed and developed in C++ using the CopperSpice libraries. CopperSpice is a stand alone C++ Gui library derived from the Qt framework. Our CopperSpice libraries can be linked directly into any standard C++11 application.

The Diamond Editor binaries are available for the following platforms:

  • Windows 32 bit
  • Windows 64 bit
  • Debian (Wheezy)
  • Fedora 20
  • Ubuntu 14.04
  • OS X 10.7
