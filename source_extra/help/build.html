Building

Diamond was compiled with the CopperSpice libraries, Hunspell Libraries using GCC, GNU Autotools, and GNU Make. To build Diamond from source all libraries and tools will need to be installed on your development system.

CopperSpice Libraries

The CopperSpice binaries (compiled version) contain the DLLs or shared libraries needed to compile and distribute your application.

  • (1)  Download the CopperSpice binary files from the download links located on the right side of our CopperSpice web site. Select the link for the desired CS version, then select source files. Pick the desired platform. CS Binary Download
  • (2)  Run the Setup program or install the package as required for the selected platform.

Hunspell Libraries

Hunspell 1.3 is the library used for spell checking.

  • For Windows the Hunspell bin, include, and lib files are included with the Diamond source. These files are only needed when building Diamond from source. The Hunspell run time DLL is supplied with the Diamond binary installer.
  • For X11 install the appropriate Hunspell package for your specific platform.
Platform Package
Debian libhunspell-dev
Fedora hunspell-devel
Ubuntu libhunspell-dev
  • For OS X use macports to install Hunspell.

Autotools & Make

Autotools is a suite of programming tools designed to assist in making source code packages portable to many Unix-like systems. The GNU build system uses Bourne-compatible shell scripts to assist the user in the configuration and build process.

The utilities used to build Diamond are listed below.

  • Autoconf version 2.67 or greater
  • Automake version 1.12.1 or greater
  • GNU Make version 3.81 or greater

For additional information about these utilities, refer to our CopperSpice documentation Building CopperSpice
