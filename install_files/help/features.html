Features
  • Edit multiple documents
    • View multiple documents by opening each one in a different tab
    • Document tabs can easily and quickly be moved around
  • Search
    • Find and replace text moving forwards or backwards in your document
    • Advanced Find can searches folders based on the a file type
    • Result for Advanced Find will display a list with file name and the line of text containing the search phrase
    • From Advanced find click on the file name to open in a new tab
  • Spell Check
    • Dictionaries for German, English, French, Italian, Portuguese, and Spanish are provided
    • New words can be added to the user dictionary
    • Dictionaries for many other languages are available online
  • Open Recent Folder
    • A list of recently opened folders is displayed under "File"
    • Click on a recent folder, Open Dialog will be displayed with the selected recent folder
    • Right click on any recent folder to delete it from the list
    • Right click on any recent folder to clear the entire list
  • Recent File List
    • A list of recently opened files is displayed under "File"
    • Click on any file to open
    • Right click on any recent file to delete it from the list
    • Right click on any recent file to clear the entire list
  • Syntax highlighting
    • Configured for more than 15 languages and file formats including C++ and Doxygen
    • User configurable by changing the Json syntax files
  • Colors
    • User definable colors for the highlight bar, foreground, and background
    • User definable colors for syntax highlighting
  • General Options
    • User definable Hot Keys
    • Specify a date and time format, used to insert the date or time in your document
  • Column mode editing
    • Switch between line mode and column mode
    • Copy, Cut, and Paste respect column and line modes
  • Macros
    • Record a set of keystrokes to replay multiple times
    • Save recorded macros
    • Load a previously recorded macro for playback
  • Window
    • Split Diamond Horizontally. Current file will be in the right pane. Tabs remain in the left pane.
    • Split Diamond Vertically. Current file will be in the lower pane. Tabs remain in the upper pane.
    • A list of the currently open tabs is shown, click on an entry to select the appropriate tab
  • Print
    • Print directly to the Printer or PDF
    • View documents before printing
    • Configurable page header and footer
  • Other Features
    • Turn on/off highlighting of the current line
    • Turn on/Off display of line numbers
    • Diamond automatically remembers the last used folder when opening a file
